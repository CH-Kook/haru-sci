GUIDE

하루사이 사용법

하루사이(Haru-Sci)는 바쁜 성인을 위한 과학 교양 학습 앱입니다. 물리학·화학·생명과학·지구과학을 하루 5분, 부담 없이 쌓아갈 수 있도록 만들었습니다. 회원가입이나 설치 없이 브라우저에서 바로 시작할 수 있습니다.

1. 오늘의 세션

홈 화면에서 "오늘의 세션 시작"

개념 카드 몇 장과 문제 몇 개를 5분 안에 끝낼 수 있는 분량으로 묶어서 매일 보여줍니다. 복습이 밀려 있으면 자동으로 오늘 세션에 섞여 나오므로, 이 버튼 하나만 눌러도 그날의 학습과 복습이 함께 끝납니다.

2. 진단으로 내 수준 맞추기

과목별 "진단" 버튼

처음 시작할 때 과목마다 짧은 적응형 진단을 통해 입문·심화·전공 세 단계 중 내 수준에 맞는 난이도를 자동으로 찾아줍니다. 진단을 건너뛰어도 언제든 나중에 다시 받을 수 있고, 단계는 탐색 화면에서 직접 바꿀 수도 있습니다.

3. 탐색: 주제별로 골라 배우기

탐색 탭

물리·화학·생명과학·지구과학 네 과목, 70개가 넘는 주제를 원하는 순서로 골라 학습할 수 있습니다. 태그로 필터링하거나 난이도 단계를 바꿔가며 관심 있는 주제부터 시작해도 됩니다.

4. 문제풀기: 수능형 문제만 몰아서

문제풀기 탭

개념 카드 없이 문제만 연속으로 풀고 싶을 때 쓰는 모드입니다. 실제 수능 기출문제가 아니라, 공개된 출제 경향을 참고해 새로 만든 수능형 문제로만 채워져 있습니다. 과목·난이도·문항 수를 고르고 바로 시작할 수 있습니다.

5. 복습은 자동으로

홈 화면 하단 "복습" 섹션

한 번 풀었던 문제는 간격 반복(SRS) 알고리즘에 따라 적절한 시점에 다시 등장합니다. 따로 복습 목록을 관리할 필요 없이, 다가오는 복습 일정과 틀렸던 문제를 홈 화면에서 확인만 하면 됩니다.

6. 이야기: 배운 것을 다른 방식으로

이야기 탭

특정 주제를 학습하거나 문제를 맞히면, 그 주제와 관련된 짧은 과학 이야기가 하나씩 열립니다. 개념을 문제로만 익히는 게 지겨울 때, 같은 내용을 다른 각도에서 다시 만나는 용도입니다.

7. 진도는 이 브라우저에만 저장됩니다

통계 탭 → 데이터 백업 · 기기 이동

별도 계정 시스템이 없어서 모든 학습 기록은 지금 쓰는 브라우저에만 저장됩니다. 다른 기기에서도 이어서 하고 싶다면 "코드로 보내기"를 눌러 나온 코드를 새 기기의 "코드로 받기"에 입력하는 게 가장 간단합니다. 코드는 10분만 유효하고 한 번 쓰면 사라집니다. 브라우저 데이터를 지우기 전에 따로 보관해두고 싶을 땐 파일로 저장하는 기능도 있습니다.

8. 홈 화면에 앱처럼 설치하기

iPhone / iPad (Safari)

하단 공유 버튼(사각형에서 위로 화살표) → "홈 화면에 추가"를 누르면 아이콘이 생기고, 그 아이콘으로 열면 브라우저 주소창 없이 전체화면 앱처럼 실행됩니다.

Android (Chrome)

주소창 오른쪽 점 세 개 메뉴 → "앱 설치" 또는 "홈 화면에 추가"를 누르면 됩니다. 인터넷이 잠깐 끊겨도 이미 열어본 내용은 계속 볼 수 있습니다.

오류나 이상한 문제를 발견했다면

문제나 이야기 화면 하단에 있는 "신고하기" 버튼을 누르면 바로 제작자에게 전달됩니다. 무엇이 잘못됐는지 간단히 적어주시면 확인 후 반영합니다.
